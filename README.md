# 高并发内存池(tcmalloc-mini)

原项目：[google-tcmalloc](https://github.com/google/tcmalloc)

---

# 项目介绍

> 这是一个CPP实现的高并发内存池，原型是谷歌的tcmalloc开源项目（Thread-Caching Malloc），既线程缓存的malloc，实现了高效的多线程内存管理，用于替代系统的内存分配相关函数（malloc、free）


# 前置需求

+ C/C++
+ 数据结构（链表、哈希桶）
+ 操作系统内存管理
+ 单例模式
+ 多线程
+ 互斥锁
+ ....
